## [0.1.0] - 2026-03-25

### Added
- Initial project scaffold generated by Project Generator
- Next.js 15 application with App Router configuration
- TypeScript strict mode configuration with comprehensive type definitions
- Tailwind CSS styling system with custom design tokens
- Supabase integration for authentication and database management
- Complete database schema with 13 tables:
  - `users` table for user authentication and profiles
  - `organizations` table for company/team data structure
  - `user_organizations` table for user-organization relationships
  - `leads` table for lead contact information and metadata
  - `lead_scores` table for historical scoring data tracking
  - `lead_activities` table for multi-channel engagement tracking
  - `scoring_rules` table for configurable scoring criteria
  - `follow_up_sequences` table for automated email sequence definitions
  - `follow_up_executions` table for sequence execution tracking
  - `integration_configs` table for API connection settings
  - `sync_jobs` table for data synchronization status monitoring
  - `reports` table for generated report data storage
  - `ai_analyses` table for AI-powered lead insights
- Row Level Security (RLS) policies for all database tables
- Authentication system with role-based access control (Owner, Analyst, Reviewer roles)
- 17 application routes from site map:
  - `/` - Public landing page
  - `/pricing` - Public pricing information
  - `/login` - Public authentication page
  - `/signup` - Public user registration
  - `/forgot-password` - Public password recovery
  - `/onboarding` - Protected user onboarding flow
  - `/dashboard` - Protected main dashboard
  - `/leads` - Protected lead management interface
  - `/leads/[id]` - Protected individual lead details
  - `/follow-up` - Protected follow-up sequence management
  - `/scoring` - Protected scoring rules configuration
  - `/integrations` - Protected integration settings
  - `/reports` - Protected analytics and reporting
  - `/settings` - Protected user and organization settings
  - `/api/webhooks/hubspot` - Public HubSpot webhook endpoint
  - `/api/webhooks/social` - Public social media webhook endpoint
  - `/api/scoring/calculate` - Protected lead scoring API
